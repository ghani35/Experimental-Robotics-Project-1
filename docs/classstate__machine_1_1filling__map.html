Detailed Description

The filling_map class.The robots keeps waiting in room E untill the whole map is loaded, the robot can know if the map is fully loaded by subscribing to the topic /map_state.

When the map is fully loaded the robot go to MOVING_IN_CORRIDORS state trough the transition move.
